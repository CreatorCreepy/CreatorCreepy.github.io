There are two types of coins in Geometry Dash: Coins and User Coins.

COINS

Coins can be gathered from mappacks and main levels. There are curently 149 coins in the game. By collectng coins you can unlock more icons.

USER COINS

User Coins are found in User Made levels. Collecting these coins (up to 1000) you can unlock achivements.
