The portals in geometry dash are what change your gamemode.

The chart below will explain what each kind of portal does:

Image Gamemode Description
Cube The green portal changes you to the normal cube gamemode.
Ship The pink/violet portal changes you to the ship gamemode.
Saw The red portal changes you to the saw gamemode.
Ufo The orange portal changes you to the ufo gamemode.
Wave The blue portal changes you to the wave gamemode.
Robot The silver portal changes you to the robot gamemode.
Spider The dark purple portal changes you to the spider gamemode.
